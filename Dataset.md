# Dataset

## Primary Evaluation Corpus (336 Generations)

10 curated open-source repositories spanning Python (Django, FastAPI) and TypeScript (Express, Next.js). 192 GPT-4o + 144 Claude 3.5 Sonnet generations across 60 tasks at three complexity levels (L1=168, L2=112, L3=56) and four prompting strategies (P1-P4). 67 validated findings after Phase 3 refinement.

---

## External Validation

### Track A: Qwen2.5-Coder-32B (HuggingFace Inference API)

36 generations (of 60 planned; API quota exhaustion) on the same 10 curated repositories, P1 prompting only. **0 findings.**

### Track B: BaxBench Django Benchmark

28 multi-file Django tasks from [LogicStar/BaxBench](https://huggingface.co/datasets/LogicStar/BaxBench). 3 models: GPT-4o (28/28), Claude Sonnet 4.5 (28/28), Qwen2.5-Coder-32B (7/28 partial). **0 findings.**

### Track C: Human Baseline (Negative Control) -- 19 Repos

375 files analyzed. **614 findings, all false positives** (monorepo workspace packages + Django internal modules).

625 files analyzed (up to 25 per repo, random sample). **880 findings** across 36/43 repos (83.7% repo-level incidence).

## Supplementary Datasets

### WebApp1K Cross-Benchmark

2,000 generations from WebApp1K: 200 scenarios x 10 models (GPT-4o variants, Claude 3.5 Sonnet, Qwen2.5-Coder-32B, DeepSeek-V3, DeepSeek-R1, Llama 405B, o1-mini, o3-mini, Gemini 2.0 Flash). P1 prompting only. **82 CFC findings** (4.1% failure rate).

### MBPP Tasks

500 tasks from Mostly Basic Python Problems. Used for task prompt calibration, not primary evaluation.
